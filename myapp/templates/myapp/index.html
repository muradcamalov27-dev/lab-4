Django Laboratoriya 4

Sadə Statik Veb Səhifə

1. Giriş

Django-nun veb inkişafındaki rolu haqqında qısa malumat. Django, Python ilə yazılmış güclü bir veb çərçivəsidir və sürətli veb tətbiqləri yaratmağa kömək edir.

2. Quraşdırma

Bu laboratoriya üçün aşağıdakılar quraşdırıldı:

  • Python 3.11.0rc1
  • Django 5.2.8
  • Django tətbiqəsi (myapp)

3. Django-nun Arxitekturası

Django Model-View-Template (MVT) arxitekturasından istifadə edir:

  • Model: Məlumatlar bazası ilə əlaqə
  • View: Məntiq və məlumatların işlənməsi
  • Template: İstifadəçiyə göstərilən HTML səhifə

4. Sadə Statik Veb Səhifənin Hazırlanması

Bu səhifə HTML şablonu, views.py faylında yazılmış view funksiyası və URL konfiqurasiyası ilə hazırlanmışdır.

  • HTML şablonu: myapp/templates/myapp/index.html
  • View funksiyası: myapp/views.py faylında index funksiyası
  • URL konfiqurasiyası: djangolab4/urls.py faylında
Qeyd: Bu laboratoriya tapşırığının 2-ci (Quraşdırma) və 4-cü (Sadə bir statik veb səhifənin hazırlanması) addımlarını əhatə edir.
